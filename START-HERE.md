# 🎮 Reverse Escape APK - দ্রুত শুরু করুন

## ⚡ মাত্র 3 মিনিটে APK পাবেন!

---

## 🔴 **STEP 1: Git কমান্ড এক্সিকিউট করুন**

আপনার টার্মিনালে এই কমান্ড চালান:

```bash
cd /workspaces/Reverse-Escape
git add .
git commit -m "Setup APK build configuration"
git push origin main
```

---

## 🟠 **STEP 2: GitHub Actions অপেক্ষা করুন**

1. GitHub এ যান: https://github.com/mdhabib9900/Reverse-Escape
2. **Actions** ট্যাবে ক্লিক করুন
3. "Build APK" ওয়ার্কফ্লো দেখবেন ⏳
4. **সবুজ চিহ্ন** ✅ হওয়া পর্যন্ত অপেক্ষা করুন (৩-৫ মিনিট)

---

## 🟢 **STEP 3: APK ডাউনলোড করুন**

### অপশন A: Artifacts থেকে (সহজ)
1. Build job খুলুন
2. **Artifacts** সেকশনে যান  
3. `Reverse-Escape-APK` ডাউনলোড করুন
4. Zip আনজিপ করুন → `app-release.apk`

### অপশন B: Releases থেকে (সরাসরি)
1. Repository > **Releases** ট্যাবে যান
2. সর্বশেষ রিলিজ খুলুন
3. `app-release.apk` সরাসরি ডাউনলোড করুন

---

## 📱 **STEP 4: ফোনে ইনস্টল করুন**

### Android ফোনে:

1. **Unknown sources চালু করুন:**
   ```
   Settings > Security > Unknown sources (ON)
   ```

2. **APK ফাইল খুলুন এবং ট্যাপ করুন:**
   - ফাইল ম্যানেজার > Downloads > app-release.apk

3. **Install বাটনে ট্যাপ করুন** ✓

4. **ইনস্টল সম্পন্ন হলে Open করুন** 

5. **খেলুন!** 🎮🎉
