```bash
cd /workspaces/Reverse-Escape

git add .

git commit -m "Setup APK build configuration"

git push origin main
```

**এটুকু!** মাত্র ৩-৫ মিনিটে আপনার APK তৈরি হবে! ✨

## 📍 তারপর কী?

### Step 1: GitHub Actions দেখুন
- যান: https://github.com/mdhabib9900/Reverse-Escape
- **Actions** ট্যাব খুলুন
- "Build APK" দেখবেন ✅

### Step 2: APK ডাউনলোড করুন
- **Artifacts** বা **Releases** থেকে ডাউনলোড করুন

### Step 3: ফোনে ইনস্টল করুন
- Settings > Security > Unknown sources (ON)
- APK ট্যাপ করুন > Install

### Step 4: খেলুন! 🎮
